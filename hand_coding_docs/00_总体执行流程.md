# SuperBizAgent 项目执行流程（精确到函数级）

## 一、项目概述

SuperBizAgent 是一个基于 LangChain + LangGraph 构建的智能运维（AIOps）系统，核心提供两大能力：
1. **RAG Agent 对话** — 基于知识库的智能问答
2. **AIOps 诊断** — 基于 Plan-Execute-Replan 模式的自动故障诊断

技术栈：FastAPI + LangChain + LangGraph + Milvus（向量库）+ DashScope（千问大模型）

## 九、LangGraph 工作流总览

### RAG Agent 工作流（隐式，由 `create_agent` 构建）

```
用户消息 → trim_messages_middleware (中间件) → LLM 推理 → 工具调用? → 工具执行 → LLM 生成回答 → 返回
```

### AIOps 工作流（显式 StateGraph）

```
ENTRY → planner → executor → replanner →┐
     ↑                                  │
     │     ┌──── continue / replan ─────┘
     │     │
     │     └──── respond → END
     │
     └──── plan 非空 → executor (循环)
```
